# YCT 汉字找不同游戏 (YCT Character Spot)

An interactive Chinese character learning game focusing on commonly confused characters for YCT (Youth Chinese Test) learners.

## Features

- **Progressive Difficulty**: Levels range from Easy to Hard with increasing grid sizes.
- **Dual Game Modes**:
  - **Solo (Robot)**: Compete against time and improve your score.
  - **PK Mode**: Play against a friend locally with real-time scoring.
- **Educational Tools**:
  - **Handwriting Canvas**: Practice writing the target character.
  - **Visual Hints**: Get help when you're stuck.
  - **Translations**: Supports both English and Mongolian translations.
- **Polished UI**: Modern, fun, and responsive design built with Tailwind CSS and Framer Motion.

## Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/yct-character-spot.git
   cd yct-character-spot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
